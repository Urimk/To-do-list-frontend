# Task Manager (Frontend)

A secure, responsive, and user-friendly task management interface built with **React** and **Tailwind CSS**.
This client application communicates with a Node.js/Express backend to provide a complete SaaS-like experience.

## Features
* **User Authentication:** Secure Login & Registration pages.
* **Protected Routes:** Unauthorized users are redirected to login; Logged-in users are redirected to the dashboard.
* **Task Management:** Create, Read, and Delete tasks with real-time updates.
* **Data Isolation:** Users can only see and manage their own tasks.
* **UX Enhancements:** Toast notifications for success/error feedback.
* **Responsive Design:** Fully adapted for mobile and desktop using Tailwind CSS.

## Run Locally

1.  **Clone the project:**
    ```bash
    git clone [https://github.com/Urimk/To-do-list-frontend.git](https://github.com/Urimk/To-do-list-frontend.git)
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

3.  **Start the app:**
    ```bash
    npm start
    ```
    *The app will run on `http://localhost:3000`*
