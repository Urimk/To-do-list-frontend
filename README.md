# Task Manager (Frontend)

A clean and responsive task management interface built with **React** and **Tailwind CSS**.
This is the client-side of the application.

**[Live Demo](https://to-do-list-frontend-lomx2dmcb-uri-ks-projects.vercel.app)** |  **[Backend Repository](https://github.com/Urimk/To-do-list-backend)**

## Tech Stack
* **Framework:** React CRA
* **Styling:** Tailwind CSS
* **State Management:** React Hooks
* **HTTP Client:** Fetch API

## Features
* Add new tasks effortlessly.
* View all tasks in a responsive list.
* Real-time updates with the backend API.

## Run Locally

1.  **Clone the project:**
    ```bash
    git clone [https://github.com/Urimk/To-do-list-frontend](https://github.com/Urimk/To-do-list-frontend)
    ```

2.  **Install dependencies:**
    ```bash
    cd folder-name
    npm install
    ```

3.  **Configure Environment:**
    Create a `.env` file and add your backend URL:
    ```env
    REACT_APP_API_URL=http://localhost:5000
    ```

4.  **Start the app:**
    ```bash
    npm start
    ```
